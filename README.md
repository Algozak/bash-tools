# Bash System Utilities

A collection of lightweight Bash utilities for Linux system administration, monitoring, and automation.

## 🛠️ Tools Catalog

* [System Overview](./SysOver) - A clean TUI dashboard showing CPU, Memory, Uptime, and Disk statistics.
* [Process Inspector](./ProcInspector) - Inspects active processes and displays internal state metrics.
* [Process ReBirth](./ProcReBirth) - A minimal process supervisor written in Bash — runs a command, watches it, and automatically restarts it if it dies.
